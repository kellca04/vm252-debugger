Phase 1 - Vm252 Debugger

Plan for Meeting Minimum Requirements:

  • Develop a user-friendly interface with input and output fields.
  • Incorporate the full range of functionalities provided by the aa, ap, amb, ba, h, mb, n, ob, q, r, s, and z commands.
  • Implement the ability to seamlessly load and execute object files.

Plan for Going Beyond Minimum Requirements:

  • Design responsive layouts that adapt to mouse position.
  • Implement a clear visual cue for breakpoints.
  • Craft a design that is both user-friendly and visually pleasing.
  • Implement all the commands at our disposal in vm252Dbg.
